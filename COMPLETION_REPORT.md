## Support & Maintenance

### Common Issues & Solutions

**Issue**: Notifications not showing
**Solution**: Check notification container in DOM, verify showNotification() calls

**Issue**: Loading spinner stuck
**Solution**: Clear cache, hard refresh (Ctrl+Shift+R)

**Issue**: Styles not applying
**Solution**: Verify style.css link, check CSS specificity, clear cache

**Issue**: API calls failing
**Solution**: Check Flask server, verify endpoints, check network tab
